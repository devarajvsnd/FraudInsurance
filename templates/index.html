Insurance Claims - Fraud Detection.

Insurance fraud is a huge problem in the industry. It's difficult to identify fraud claims. The project aims to help the Auto Insurance industry with this problem. With some auto insurance data as a input the project demonstrates how to create a predictive model, that predicts if an insurance claim is fraudulent or not. This will be a Binary Classification task. The project is implemented with continious integration and continious deployment. The models used for the training can be updated and the Insurance claims can be validated with a single input and Bulk prediction as well.
